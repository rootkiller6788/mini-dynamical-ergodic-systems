# mini-symbolic-dynamics

Symbolic dynamics, shifts of finite type, subshifts, zeta function

## Structure
| Layer | Files | Description |
|-------|-------|-------------|
| Core | 3 | Basic definitions, laws, objects |
| Morphisms | 3 | Homomorphisms, isomorphisms, equivalences |
| Constructions | 4 | Products, quotients, subobjects, universal |
| Properties | 3 | Invariants, preservation, classification |
| Theorems | 4 | Key theorems (sorry proofs) |
| Examples | 2 | Standard examples, counterexamples |
| Bridges | 4 | To algebra, topology, geometry, computation |

## Quick Start
```bash
lake build
lake env lean --run Test/Smoke.lean
```
